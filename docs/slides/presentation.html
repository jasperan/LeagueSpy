Riot's API has strict rate limits and requires developer approval. Third-party sites show stats, but they don't push notifications. You want to know the moment your friend loses another ranked game so you can flame them properly.

The Solution: LeagueSpy

A Discord bot that scrapes leagueofgraphs.com with a stealth browser, stores matches in Oracle Database, and announces every game in your server. Wins get a green embed. Losses get roasted in Spanish by a local LLM.

Stealth Scraper

  • _extract_duration (time string)
  • Match details: 10 players + bans

Slash Commands

21

Key Design Decisions

Embed Before Insert

Discord embed sent before DB insert. Lost announcements are worse than duplicates.

Deque Decoupling

new_matches deque (maxlen=100) decouples detection from roast generation.

Zero-Cost Rivalry

Just a DB query on match_id after insert. No extra scraping needed.

Graceful Degradation

vLLM down? Roasts skip silently. Match details fail? Basic embed still fires.

KDA: inf on 0 Deaths

float("inf") instead of division error. Clean, intentional edge case handling.

Thinking Disabled

enable_thinking: false for Qwen3.5. Faster, punchier roast output.
