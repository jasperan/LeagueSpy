// system overview

LeagueSpy

A Discord bot that monitors League of Legends match history for your friend group, including smurf accounts, and announces results in real time.

01 // Pipeline

How It Works

Config
config.yaml
Summoner list with player names, slugs, regions, and smurf aliases.
→
Scheduler
tasks.loop (5 min)
Discord.py background task fires the scraper on a fixed interval.
→
Scraper
scrapling / op.gg
StealthyFetcher bypasses anti-bot protections to pull match data from op.gg.
↓
Announcer
Discord Bot
Posts rich embeds (green for wins, red for losses) with champion, KDA, duration, and mode.
←
Storage
Oracle Database
Persists match records. Diff check compares stored IDs against scraped IDs to detect new games.
